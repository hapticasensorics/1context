# Talk Conventions - Your Context

Your Context is durable orientation, so its talk page should be slower and more
evidence-heavy than For You.

Use one timestamped markdown file per contribution. Each top-level file is one
thread. Replies use `parent:` frontmatter pointing to the parent filename or
stem.

## What Belongs Here

- Proposed working-style observations.
- Repeated preferences or corrections that should guide future collaborators.
- Standing requests the operator has made explicitly.
- Evidence that a pattern is stable enough to preserve.
- Disagreements, redactions, and reversals when an observation is not true.

## Promotion Rule

Do not turn a single weak signal into permanent personality. Prefer repeated
evidence, an explicit user instruction, or a strong correction from experience.

## Archive Policy

Keep active talk entries in this folder for 90 days. After that, move settled
conversation files into `archive/` while preserving filenames, frontmatter, and
parent/thread references.
