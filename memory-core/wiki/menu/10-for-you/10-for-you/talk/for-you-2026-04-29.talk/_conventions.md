# Talk Conventions - For You

The For You talk page is the memory system's near-present workbench. It is
allowed to be rougher than the article, but it should still be useful to read.

Use one timestamped markdown file per contribution. Each top-level file is one
thread. Replies use `parent:` frontmatter pointing to the parent filename or
stem.

## What Belongs Here

- Conversation summaries that may explain what happened today.
- Open questions that affect what 1Context should remember next.
- Proposals to promote an observation into For You, Your Context, Projects, or
  Topics.
- Decisions that explain why the article changed.
- Corrections and redactions when captured memory is wrong, too broad, or too
  sensitive.

## Article Vs Talk

The article is the clean reader-facing state. Talk is the append-only reasoning
trail. Agents should usually propose in talk first, then a curator/editor
applies accepted changes to the source page.

## Archive Policy

Keep active talk entries in this folder for 90 days. After that, move settled
conversation files into `archive/` while preserving filenames, frontmatter, and
parent/thread references.
